# Layori-Soetan-NXU-BAN-6420-Module2-Assignment-Salary-Function

## Overview
This project utilizes Python and R to process employee salary data, encompassing data import, dictionary processing, error handling, CSV export, and zip file management.

## How to Use

### Python (Jupyter Notebook)
1. Open `employee_notebook.ipynb`.
2. Run all cells to:
   - Import salary data
   - Search for employee details using their name 
   - Export to CSV and zip employee details 

### R (Jupyter Notebook)
1. Open `Rscript_unzip.ipynb`
2. Run all cells to:
   - Unzip and list files
   - Read the CSV file
   - Display the first row
